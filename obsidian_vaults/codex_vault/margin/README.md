# Wholesale Lender Economics, Using Dummy Numbers

This packet explains how a UWM-like wholesale mortgage lender can make more or less money on the same basic borrower transaction.

It uses dummy data only. The point is to understand the economic machine, not to estimate any specific company's real margin.

## The One-Sentence Version

A wholesale lender makes money when the total value it can create from a funded mortgage is greater than the money it gives up to win the loan, compensate the broker, hedge the lock, fund the loan, process it, sell it, and service it.
