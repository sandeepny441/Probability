# Mortgage Screening Lists

This folder explains the main screening buckets a mortgage lender or broker may touch before, during, and after a loan.

It is written for intuition first. Compliance teams should still rely on their own policies, counsel, investor guides, and vendor procedures.

## The One-Sentence Version

Mortgage screening is not one list. It is a control system that asks:

```text
Are we legally allowed to transact with this person?
Is this person or company eligible for this loan program or investor?
Does the file contain identity, property, income, occupancy, or transaction fraud risk?
Did anything change after closing that we must report, block, cure, or monitor?
```
