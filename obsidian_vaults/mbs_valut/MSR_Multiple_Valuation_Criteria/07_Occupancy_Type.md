# Occupancy Type

## Prepay & Default Profiles
- **Primary Residence**: Baseline behavior. Steady, moderate prepay.
- **Second Home**: Slightly faster prepay (financial flexibility), moderate default.
- **Investment Property**: Higher LLPAs at origination, higher default in stress, but often higher rate = attractive servicing fee. Prepay behavior linked to cash-flow economics.

## Multiple Impact
- Primary residence pools: standard/premium multiples.
- Investor-heavy pools: discounted for default risk but note-rate premium partially offsets.
