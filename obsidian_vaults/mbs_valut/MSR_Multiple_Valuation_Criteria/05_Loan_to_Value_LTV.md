# Loan-to-Value (LTV)

Original and current (mark-to-market) LTV affect prepay ability and default severity.

## Prepay Effect
- **Low LTV (< 60%)**: Easy to refinance, cash-out capable → faster prepay when ITM.
- **High LTV (> 95%)**: Harder to refinance (need appraisal, MI), slower prepay → higher multiple.
- **Underwater (LTV > 100%)**: Very slow prepay, but higher default risk.

## Default & Severity Effect
- Higher LTV → higher LGD (Loss Given Default).
- Affects advance obligations and cost to service delinquent loans.

## Mark-to-Market LTV
- Updated using HPI (Home Price Index) at MSA level.
- Rising HPI improves borrower equity → increases refi risk.
- Falling HPI increases default risk.
