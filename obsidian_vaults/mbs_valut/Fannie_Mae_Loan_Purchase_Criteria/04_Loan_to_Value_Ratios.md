# Loan-to-Value Ratios (LTV / CLTV / HCLTV)

Measures loan amount relative to property value.

## Definitions
- **LTV**: 1st mortgage ÷ property value
- **CLTV**: (1st + 2nd liens) ÷ value
- **HCLTV**: (1st + full HELOC line) ÷ value

## Maximum LTV by Product
- **Primary residence, purchase, fixed-rate 1-unit**: 97% LTV (HomeReady/Standard 97)
- **Primary residence, 2-4 unit**: 85–95%
- **Second home**: 90% LTV
- **Investment property**: 85% purchase (1-unit); 75% (2-4 unit)
- **Cash-out refinance**: 80% (primary 1-unit)

## Notes
- LTV > 80% requires [[11_Mortgage_Insurance]].
- Higher LTV = higher [[19_Pricing_and_LLPAs]].
- Value = lesser of appraised value or purchase price.
