# Documentation Requirements

Full documentation is required — no "stated income" / "no-doc" loans.

## Required Documents
- **1003 Uniform Residential Loan Application (URLA)**
- **Income**: Paystubs, W-2s, tax returns, VOEs
- **Assets**: 2 months bank statements, retirement statements
- **Credit**: Tri-merge credit report
- **Identity**: Government ID, SSN verification
- **IRS Form 4506-C**: Authorizes tax transcript retrieval
- **Purchase contract** (for purchases)
- **Title commitment / preliminary title report**
- **Homeowner's insurance binder**
- **Flood certification** (and flood insurance if in SFHA)
- **Appraisal report**
- **Closing Disclosure (CD)**

## Loan File Retention
- Lenders must retain the loan file for the life of the loan + years, per Selling Guide.
